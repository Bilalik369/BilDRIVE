# نظام نسيت كلمة المرور - BilDRIVE

## نظرة عامة
تم تنفيذ نظام كامل لنسيت كلمة المرور في تطبيق BilDRIVE مع دعم كامل للواجهة الأمامية والخلفية.

## الميزات المنجزة

### 🔐 **صفحة نسيت كلمة المرور**
- نموذج لإدخال البريد الإلكتروني
- التحقق من صحة البريد الإلكتروني
- إرسال رابط إعادة التعيين
- إمكانية إعادة إرسال الرابط

### 🔗 **صفحة إعادة تعيين كلمة المرور**
- نموذج لإدخال كلمة المرور الجديدة
- تأكيد كلمة المرور
- التحقق من قوة كلمة المرور
- رسالة نجاح بعد إعادة التعيين

### 📧 **رسائل البريد الإلكتروني**
- تصميم احترافي متناسق مع العلامة التجارية
- دعم اللغة العربية
- روابط تعمل مع الواجهة الأمامية
- انتهاء صلاحية الروابط بعد ساعة واحدة

### 🎨 **التصميم**
- نفس الألوان والتصميم المستخدم في باقي النماذج
- دعم Tailwind CSS
- تجاوب كامل مع جميع الأجهزة
- رسائل خطأ ونجاح واضحة

## المسارات المتاحة

### الواجهة الأمامية (Frontend)
```
/auth/forgot-password          - صفحة نسيت كلمة المرور
/auth/reset-password/:token    - صفحة إعادة تعيين كلمة المرور
```

### الخلفية (Backend)
```
POST /api/auth/forgot-password     - إرسال رابط إعادة التعيين
POST /api/auth/reset-password/:token - إعادة تعيين كلمة المرور
```

## كيفية الاستخدام

### 1. طلب إعادة تعيين كلمة المرور
1. انتقل إلى صفحة تسجيل الدخول
2. انقر على "نسيت كلمة المرور؟"
3. أدخل بريدك الإلكتروني
4. انقر على "إرسال تعليمات إعادة التعيين"
5. تحقق من بريدك الإلكتروني

### 2. إعادة تعيين كلمة المرور
1. انقر على الرابط في البريد الإلكتروني
2. أدخل كلمة المرور الجديدة
3. أكد كلمة المرور الجديدة
4. انقر على "إعادة تعيين كلمة المرور"
5. انتقل إلى صفحة تسجيل الدخول

## الأمان

### 🔒 **ميزات الأمان**

## التخصيص

### الألوان والتصميم
يمكن تخصيص الألوان من خلال ملف `tailwind.config.js`:

```javascript
module.exports = {
  theme: {
    extend: {
      colors: {
        primary: '#9929EA',
        'text-dark': '#1f2937',
        'text-secondary': '#6b7280',
        'bg-main': '#f9fafb',
        'border-color': '#e5e7eb',
      }
    }
  }
}
```

### رسائل البريد الإلكتروني
يمكن تخصيص رسائل البريد الإلكتروني في `backend/utils/email.utils.js`

## استكشاف الأخطاء

### مشاكل شائعة
1. **البريد الإلكتروني لا يصل**: تحقق من إعدادات SMTP
2. **الرابط لا يعمل**: تحقق من متغير البيئة `FRONTEND_URL`
3. **خطأ في التحقق**: تحقق من صحة الرمز وانتهاء الصلاحية

### متغيرات البيئة المطلوبة
```env
EMAIL_HOST=smtp.gmail.com
EMAIL_PORT=587
EMAIL_SECURE=false
EMAIL_USER=your_email@gmail.com
EMAIL_PASS=your_app_password
EMAIL_FROM=your_email@gmail.com
FRONTEND_URL=http://localhost:3000
```

## الاختبار

### اختبار النظام
1. انتقل إلى `/auth/forgot-password`
2. أدخل بريد إلكتروني صحيح
3. تحقق من وصول البريد الإلكتروني
4. انقر على الرابط
5. أدخل كلمة مرور جديدة
6. تحقق من نجاح العملية
